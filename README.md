<code important-text-pink>vite-fast-template</code> 是一个基于 [Vue](https://github.com/vuejs/vue)、[Vite](https://github.com/vitejs/vite)、[Markdown](https://www.markdownguide.org/) 的快速启动模板

Author: [@JoiePink | Joie Pink]

Repository: https://github.com/JoiePink/vite-fast-template

Technology Stack: {Vue} {Vite} {Markdown} {UnoCSS} {Iconify} {TypeScript}

Special:

- **快速启动**：基于 `Vite + Vue 3 + Markdown` 的项目脚手架，开箱即用，适合个人/团队快速搭建博客或文档站。
- **VueUse 自动导入**：内置 `VueUse`，常用 hooks 可自动导入，无需手动 `import`，让组合式 API 的书写更加顺手。
- **文件即路由**：集成 `vue-router`，`pages` 目录下的 `.vue` / `.md` 文件会自动生成路由，零配置即可使用。
- **原子化样式**：内置 `UnoCSS + Iconify`，通过原子类名与图标类名即可完成大部分样式与图标开发。
- **Markdown 即页面**：Markdown 文件可以直接作为页面渲染，内置目录（`[[toc]]`）、锚点、链接增强、代码高亮等多种扩展能力。
- **依赖版本管理**：通过 `pnpm catalog` 对依赖进行分类管理，不同子项目/场景的依赖版本边界更清晰、可控。
